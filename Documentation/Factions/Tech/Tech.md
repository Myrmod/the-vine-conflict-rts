# Tier 1 tech:

## The Amuns:
- starting building: power plant (obelisk) - provides power and buffs/upgrades
- power plant unlocks -> prism, siphon, barracks, airfield
- prism must be placed on resource spawner (max one prism per spawner)
- rax/airfield unlocks -> factory, naval yard
- factory unlocks -> t2 tech structure or mcv upgrade to be determined

## The Radix:
- starting building: heart ( no power for this faction )
- heart produces seedlings used for creep spread and structure construction
- seedling is consumed on successful completion of spread/build action
- if controlling player interrupts before completion, seedling is not consumed
- heart unlocks-> vine spreader, barracks, root conduit, need to be built on creep
- root conduit unlocks--> nothing
- rax unlocks -> factory, airfield, naval yard
- factory, airfield, naval yard unlock -> tech structure/upgrade building

## The Legion:
- starting building : power plant
- power plant unlocks-> barracks and ref
- ref unlocks -> factory
- barracks unlocks -> factory
- factory unlocks -> airfield, t2 structure. navy
- airfield unlocks -> nothing

## The Remnants:
- starting building: power plant
- power plant unlocks-> barrack, factory
- rax, factory -> unlocks airfield, navy
- factory unlocks -> t2 structure
