# 1\. Core Premise

A global-scale RTS set on a future Earth overrun by alien-introduced hyper-growth vines.

The vines are:

*   Not sentient
    
*   Self-propagating planetary biomass
    
*   The main terrain modifier and economic backbone
    
*   A shared system every faction interacts with differently
    

All factions have **infinite income scaling**, but through incompatible relationships with vines.

No faction is morally right â€” only strategically consistent.
